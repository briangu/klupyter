# KlongPy Jupyter Kernel

Use (KlongPy)[http://klongpy.org) in Jupyter Notebooks! 

# Install

```bash
jupyter kernelspec install --user klongpy_kernel
```

# Running (locally) 

```bash
jupyter console --kernel klongpy_kernel
```

or

```bash
jupyter notebook
```
